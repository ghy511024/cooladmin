js 初始化表格

    <script>
            $(document).ready(function () {
                $('#table').dataTable();
            })
       </script>

PS:需要注意一点：有些表格中的一些按钮需要绑定事件（click 之类的），这些事件最好在初始化表格之前绑定，要不然在翻页时事件会失效


    <script>
            $(document).ready(function () {
                $(".xxx").on("click",function(){})
                ...
                ...
                //事件绑定完了初始化表格插件
                $('#table').dataTable();
            })
       </script>
