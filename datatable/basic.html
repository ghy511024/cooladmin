html 代码

   <table id="example" class="table table-bordered">
       <thead>
        ....
        </thead>
       <tbody>
        ....
        </tbody>
    </table>
js 初始化表格（如果引用了 入口文件将可不写初始化方法，但table的id为example）

    <script>
            $(document).ready(function () {
                $('#example').dataTable();
            })
       </script>

PS:需要注意一点：有些表格中的一些按钮需要绑定事件（click 之类的），这些事件最好在初始化表格之前绑定，要不然在翻页时事件会失效


    <script>
            $(document).ready(function () {
                $(".xxx").on("click",function(){})
                ...
                ...
                //事件绑定完了初始化表格插件
                $('#example').dataTable();
            })
       </script>
