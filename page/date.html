日期插件（jquery ui）
版本 jquery jquery-ui.js jquery-ui.css
章鱼tv 现用版本 http://static.ws.kukuplay.com/common/lib/jquery/v1.9.2/jquery-1.9.2.js http://static.ws.kukuplay.com/common/lib/jquery-ui/v1.8.21/jquery-ui.1.8.21.min.js http://static.ws.kukuplay.com/common/lib/jquery-ui/v1.8.21/jquery-ui.1.8.21.min.css

1：本文版本 为章鱼后端通用版本，中文设置已经集成到类库中，无需多引用东西。有良好的兼容性,推荐使用。

例子


        <script type="text/javascript">
            $(document).ready(function () {
                $("#date2").datetimepicker();//同时选定日期和时间
            })
        </script>

限制某一段时间不可选


        <script type="text/javascript">
            $(document).ready(function () {
                $("#date2").datetimepicker({//月份中数值需要加1（0代表1月）
                    minDate: new Date(2015, 7, 12),
                    maxDate: new Date(1440518400000)
                });
            })
        </script>

设置默认时间


        <script type="text/javascript">
             $("#date3").datetimepicker({
                    defaultDate: "2015-12-13"
                });
        </script>
