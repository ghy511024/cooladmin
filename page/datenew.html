日期插件（jquery ui）
版本 jquery jquery-ui.js jquery-ui.css
jquery ui 最新版 http://static.ws.kukuplay.com/common/lib/jquery/v1.11.3/jquery-1.11.3.min.js http://static.ws.kukuplay.com/common/date/v2.3.7/jquery.datetimepicker.min.js http://static.ws.kukuplay.com/common/date/v2.3.7/jquery.datetimepicker.css

1：最新版的的中文设置只需要调节一个参数就行，详见官网

2：新版官网，github

例子


        <script type="text/javascript">
            $(document).ready(function () {
                $("#date2").datetimepicker();//同时选定日期和时间
            })
        </script>

限制某一段时间不可选


        <script type="text/javascript">
            $(document).ready(function () {
                $("#date2").datetimepicker({//月份中数值需要加1（0代表1月）
                    minDate: new Date(2015, 7, 12),
                    maxDate: new Date(1440518400000)
                });
            })
        </script>

新版的没有花太多时间去适配，兼容性、中文支持等做的都不是很好，所以章鱼后端目前采用的还是jquery 通用版的，那个版本做过适配有良好的，中文支持和兼容性。
